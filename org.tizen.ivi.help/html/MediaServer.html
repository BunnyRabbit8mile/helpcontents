1. Interfaces

1.1. MediaServerManagerObject

This interface defines what is instantiated by the Tizen object by the Tizen Platform. There will be a tizen.mediaserver object that allows access to the functionality of the MediaServerManager API.
[NoInterfaceObject] interface MediaServerManagerObject {
    readonly attribute MediaServerManager mediaserver;
};

Tizen implements MediaServerManagerObject;

1.2. MediaServerManager

The MediaServerManager interface defines access to local network media servers discovery functionality. 
[NoInterfaceObject] interface MediaServerManager  { 

    void scanNetwork(MediaServerFoundServerCallback successCallback, optional MediaServerErrorCallback errorCallback);

};

Methods

scanNetwork
Scans local network for media servers.
void scanNetwork(MediaServerFoundServerCallback successCallback, optional MediaServerErrorCallback errorCallback);

Parameters:

  • successCallback: The method to call when each MediaServer is discovered.
  • errorCallback[optional]: The method to call when no MediaServer is discovered.

Exceptions:

  • WebAPIError

    • with error type UnknownError, if no MediaServer is discovered.

1.3. MediaServer

The MediaServer interface. 
[NoInterfaceObject] interface MediaServer {

    readonly attribute DOMString id;

    readonly attribute DOMString friendlyName;

    readonly attribute object root;
    
    void browse(DOMString containerId, DOMString sortMode, unsigned long count, unsigned long offset, MediaServerBrowseFindCallback successCallback, optional MediaServerErrorCallback errorCallback);

    void find(DOMString containerId, DOMString searchFilter, DOMString sortMode, unsigned long count, unsigned long offset, MediaServerBrowseFindCallback successCallback, optional MediaServerErrorCallback errorCallback);

};

Attributes

  • readonly DOMString id
    An opaque unique ID.
  • readonly DOMString friendlyName
    A user-friendly name for the device.
  • readonly object root
    Media server root folder.

Methods

browse
Browses for media items under a given media container.
void browse(DOMString containerId, DOMString sortMode, unsigned long count, unsigned long offset, MediaServerBrowseFindCallback successCallback, optional MediaServerErrorCallback errorCallback);

Parameters:

  • containerId: A unique identifier for media container in a media server.
  • sortMode: Sort mode for media items.
  • count: The maximum number of media items to return out of the browse result (0 for no limit).
  • offset: The (0-based) index of the first item in the browse result.
  • successCallback: The method to call when media items are discovered.
  • errorCallback[optional]: The method to call when an error occurs.

Exceptions:

  • WebAPIError

    • with error type UnknownError, if an error occurs.

find
Searches for media items under a given MediaContainer.
void find(DOMString containerId, DOMString searchFilter, DOMString sortMode, unsigned long count, unsigned long offset, MediaServerBrowseFindCallback successCallback, optional MediaServerErrorCallback errorCallback);

Parameters:

  • containerId: A unique identifier for media container in a media server.
  • searchFilter: The search criteria string.
  • sortMode: Sort mode for media items.
  • count: The maximum number of media items to return out of the browse result (0 for no limit).
  • offset: The (0-based) index of the first item in the browse result.
  • successCallback: The method to call when media items are discovered.
  • errorCallback[optional]: The method to call when an error occurs.

Exceptions:

  • WebAPIError

    • with error type UnknownError, if an error occurs.

1.4. MediaServerError

The MediaServerError interface defines error codes and messages.
[NoInterfaceObject] interface MediaServerError {

    const unsigned short GENERIC = 1;

    attribute unsigned short code;

    attribute DOMString message;

};

Constants

  • GENERIC
    It defines an error code.

Attributes

  • unsigned short code
    An attribute to get error code.
  • DOMString message
    An attribute to get error message.

1.5. MediaServerFoundServerCallback

This interface invokes the success callback that is invoked when a MediaServer is discovered. 
[Callback=FunctionOnly, NoInterfaceObject] interface MediaServerFoundServerCallback {

    void onsuccess(MediaServer value);

};

Methods

onsuccess
The callback function used to return a MediaServer object. Called when the media server is found successfully. 
void onsuccess(MediaServer value);

Parameters:

  • value: A MediaServer object.

1.6. MediaServerBrowseFindCallback

This interface specified a success callback that is invoked when some media items is found.
[Callback=FunctionOnly, NoInterfaceObject] interface MediaServerBrowseFindCallback {

    void onsuccess(object[] value);

};

Methods

onsuccess
The callback function used to return a list of MediaServerItem objects. Called when the list of media server item is retrieved successfully.
void onsuccess(object[] value);

Parameters:

  • value: A list of media items.

1.6. MediaServerErrorCallback

This interface specified a failure callback that is invoked when any errors are happened.
[Callback=FunctionOnly, NoInterfaceObject] interface MediaServerErrorCallback {

    void onfailure(MediaServerError error);

};

Methods

onfailure
The callback function used to return a MediaServerError object. Called when an error occurs.
void onfailure(MediaServerError error);

Parameters:

  • error: A MediaServerError object.

    };

        attribute unsigned short code;

        attribute DOMString message;

    };

    [Callback=FunctionOnly, NoInterfaceObject] interface MediaServerFoundServerCallback {

        void onsuccess(MediaServer value);

    };

    [Callback=FunctionOnly, NoInterfaceObject] interface MediaServerBrowseFindCallback {

        void onsuccess(object[] value);

    };

    [Callback=FunctionOnly, NoInterfaceObject] interface MediaServerErrorCallback {

        void onfailure(MediaServerError error);

    };
};
