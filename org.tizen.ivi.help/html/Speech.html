Speech API

The Web Speech API aims to enable web developers to provide, in a web browser, speech-input and text-to-speech output features that are typically not available when using standard speech-recognition or screen-reader software.

2.3. SpeechRecognition

[Constructor()]

interface SpeechRecognition : EventTarget {

    attribute SpeechGrammarList grammars;

    attribute DOMString lang;

    attribute boolean continuous;

    attribute boolean interimResults;

    attribute unsigned long maxAlternatives;

    attribute DOMString serviceURI;

    void start();

    void stop();

    void abort();

    attribute EventHandler onaudiostart;

    attribute EventHandler onsoundstart;

    attribute EventHandler onspeechstart;

    attribute EventHandler onspeechend;

    attribute EventHandler onsoundend;

    attribute EventHandler onaudioend;

    attribute EventHandler onresult;

    attribute EventHandler onnomatch;

    attribute EventHandler onerror;

    attribute EventHandler onstart;

    attribute EventHandler onend;

};

Attributes

  • SpeechGrammarList grammars
    An attribute to stores the collection of SpeechGrammar objects which represent the grammars that are active for this recognition.
  • DOMString lang
    An attribute to set the language of the recognition for the request.
  • DOMString continuous
    The default value is false.
  • boolean interimResults
    An attribute to control whether interim results are returned. The default value is false.
  • unsigned long maxAlternatives
    An attribute to set the maximum number of SpeechRecognitionAlternatives per result. The default value is 1.
  • DOMString serviceURI
    An attribute to specifies the location of the speech recognition service that the web application wishes to use.
  • EventHandler onaudiostart
    Fired when the user agent has started to capture audio.
  • EventHandler onsoundstart
    Fired when some sound, possibly speech, has been detected.
  • EventHandler onspeechstart
    Fired when the speech that will be used for speech recognition has started.
  • EventHandler speechend
    Fired when the speech that will be used for speech recognition has ended. The speechstart event must always have been fire before speechend.
  • EventHandler soundend
    Fired when some sound is no longer detected. The soundstart event must always have been fired before soundend.
  • EventHandler audioend
    Fired when the user agent has finished capturing audio. The audiostart event must always have been fired before audioend.
  • EventHandler onresult
    Fired when the speech recognizer returns a result.
  • EventHandler onnomatch
    Fired when the speech recognizer returns a final result with no recognition hypothesis that meet or exceed the confidence threshold.
  • EventHandler onerror
    Fired when a speech recognition error occurs.
  • EventHandler onstart
    Fired when the recognition service has begun to listen to the audio with the intention of recognizing.
  • EventHandler onend
    Fired when the service has disconnected.

Methods

start
When the start method is called it represents the moment in time the web application wishes to begin recognition. Once the system is successfully listening to the recognition the user agent must raise a start event.
void start();

Exceptions:

  • WebAPIException

    • with error type InvalidStateError, if the start method is called on an already started object.

stop
The stop method represents an instruction to the recognition service to stop listening to more audio, and to try and return a result using just the audio that it has already received for this recognition.
void stop();
abort
The abort method is a request to immediately stop listening and stop recognizing and do not return any information but that the system is done. When the abort method is called, the speech service must stop recognizing. The user agent must raise an end event once the speech service is no longer connected.
void abort();

2.5. SpeechRecognitionAlternative

The SpeechRecognitionAlternative represents a simple view of the response that gets used in a n-best list.
[NoInterfaceObject] interface SpeechRecognitionAlternative {

    readonly attribute DOMString transcript;

    readonly attribute float confidence;

};

Attributes

  • readonly DOMString transcript
    An attribute to get the raw words that the user spoke.
  • readonly float confidence
    An attribute to get a numeric estimate between 0 and 1 of how confident the recognition system is that the recognition is correct. A higher number means the system is more confident.

2.6. SpeechRecognitionResult

The SpeechRecognitionResult object represents a single one-shot recognition match, either as one small part of a continuous recognition or as the complete return result of a non-continuous recognition.
[NoInterfaceObject] interface SpeechRecognitionResult {

    readonly attribute unsigned long length;

    SpeechRecognitionAlternative item(unsigned long index);

    readonly attribute boolean final;

};

Attributes

  • readonly unsigned long length
    An attribute to represent how many n-best alternatives are represented in the item array.
  • readonly boolean final
    An attribute to control whether or not an 3 characters. interim result cannot be changed.

Methods

item
Returns a SpeechRecognitionAlternative from the index into an array of n-best values. If index is greater than or equal to length, returns null.
SpeechRecognitionAlternative item(unsigned long index);

Parameters:

  • index: An index of the SpeechRecognitionAlternative array.

Return value:

A SpeechRecognitionAlternative object.

2.7. SpeechRecognitionResultList

The SpeechRecognitionResult object represents a single one-shot recognition match, either as one small part of a continuous recognition or as the complete return result of a non-continuous recognition.
[NoInterfaceObject] interface SpeechRecognitionResultList {

    SpeechRecognitionResult item(unsigned long index);

Attributes

Methods

item
Returns a SpeechRecognitionResult from the index into an array of result values. If index is greater than or equal to length, returns null.
SpeechRecognitionResult item(unsigned long index);

Parameters:

Return value:

2.8. SpeechRecognitionEvent

The SpeechRecognitionEvent is the event that is raised each time there are any changes to interim or final results.
[NoInterfaceObject] interface SpeechRecognitionEvent : Event {

    readonly attribute unsigned long resultIndex;

    readonly attribute SpeechRecognitionResultList results;

    readonly attribute any interpretation;

    readonly attribute Document emma;

};

Attributes

  • readonly unsigned long resultIndex
    An attribute to get the lowest index in the results array that has changed.
  • readonly SpeechRecognitionResultList results
    An attribute to get all current recognition results for this session.
  • readonly any interpretation
    An attribute to represent the semantic meaning from what the user said.
  • readonly Document emma
    An attribute to get emma.

2.9. SpeechGrammar

The SpeechGrammar object represents a container for a grammar.
[Constructor()]

interface SpeechGrammar {

    attribute DOMString src;

    attribute float weight;

};

Attributes

  • DOMString src
    An attribute to get the URI for the grammar.
  • float weight
    An attribute to control the weight that the speech recognition service should use with this grammar. By default, a grammar has a weight of 1.

2.10. SpeechGrammarList

The SpeechGrammarList object represents a collection of SpeechGrammar objects.
[Constructor()]

interface SpeechGrammarList {

    readonly attribute unsigned long length;

    SpeechGrammar item(unsigned long index);

    void addFromURI(DOMString src, optional float weight);

    void addFromString(DOMString string, optional float weight);

};

Attributes

  • readonly unsigned long length
    An attribute to represent how many grammars are currently in the array.

Methods

item
Returns a SpeechGrammar from the index into an array of grammars.
SpeechGrammar item(unsigned long index);

Parameters:

  • index: An index of the SpeechGrammar array.

Return value:

A SpeechGrammar object.
addFromURI
Appends a grammar to the grammars array parameter based on URI.
void addFromURI(DOMString src, optional float weight);

Parameters:

  • src: The URI for the grammar.
  • weight[optional]: The weight the grammar.
addFromString
Appends a grammar to the grammars array parameter based on text.
void addFromString(DOMString string, optional float weight);

Parameters:

  • string: The string for the grammar.
  • weight[optional]: The weight the grammar.
